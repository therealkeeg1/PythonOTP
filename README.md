# PythonOTP
This is a simple OTP (One Time Pad) encryption program for encrypting/decrypting text.
This is writen in python 2.7 it will not work in python3!


### Examples

Basic encrypting
```
python OneTimePad.py -e -da "Wow this really works does it?"
```
